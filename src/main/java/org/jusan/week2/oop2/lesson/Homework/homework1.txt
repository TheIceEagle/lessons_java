Создать плейлист, добавить в него песни и проиграть их.

Сущности:

1) Плейлист (Playlist) – контейнер, который будет содержать все песни
– свойство “songList” – список доступных песен – коллекция ArrayList
– метод “addSong” – добавить новую песню в songList
– метод “playAllSongs” – проигрывает поочередно все песни Song методом play

2) Песня (Song) – объект, который будем добавлять в коллекцию songList
– свойство “name”: название песни – String
– свойство “author”: ссылка на объект Author
– метод “play”: проиграть текущую песню – выводит в консоль: название песни и имя автора

3) Автор песни (Author) – можно использовать обычный класс (без интерфейса), который условно называют POJO (Plain Old Java Object) – просто содержит свойства без методов
– свойство “name”: имя – String

При проигрывании песни должно выводиться: “Играет песня название, автор автор“

Запуск
Создаем объекты:

плейлист
любое количество объектов Song
Добавляем созданные песни Song поочередно в плейлист с помощью метода addSong.
Проигрываем все песни из плейлиста методом playAllSongs (который с помощью цикла проходит по всем элементам коллекции songList и вызывает у каждой песни Song метод play)
